# Добавление фотографий

Чтобы добавить детские фотографии жениха и невесты:

1. Добавьте фотографию жениха в детстве в этот каталог с именем `groom-child.jpg` (или `.png`)
2. Добавьте фотографию невесты в детстве с именем `bride-child.jpg` (или `.png`)
3. Обновите `components/PhotoScreen.tsx`:
   - Установите `hasGroomPhoto = true`
   - Установите `hasBridePhoto = true`

**Рекомендации по изображениям:**
- Формат: JPG или PNG
- Размер: рекомендуется квадратные изображения 512x512px или больше
- Оптимизируйте изображения для веба перед загрузкой

После добавления фотографий и обновления флагов, изображения будут автоматически отображаться в приложении.
